If you see this page, the Flask web application is working.
Now you could try to make POST request with curl in terminal, something like this:
curl -XPOST --insecure -d'{"word":"thumbs_up", "count": 5}' https://myvm.localhost/
or
curl -XPOST -d'{"word":"elephant", "count": 3}' http://192.168.1.246/

The output should be:
👍thumbs_up👍thumbs_up👍thumbs_up👍thumbs_up👍thumbs_up👍
and
🐘elephant🐘elephant🐘elephant🐘
respectively

NOTE: in case you are using an https (secure) connection, you should add the --insecure flag to allow curl connect to the server using an untrusted self-signed certificate. More details here: https://curl.se/docs/sslcerts.html

Also you could see preview of emoji by appending it's name in address line.
